# Day 4 - Helpdesk Ticket Routing in Make

## Project
Employee Helpdesk Ticket Routing

## Scenario
Built a Make scenario where employees submit a Google Form for IT support requests. The ticket details are logged into Google Sheets, and a Router is used to direct requests to different paths based on conditions.

## Workflow
Google Forms


Google Sheets (Log Ticket)


Router

Low Priority

Medium Priority

High Priority

## Concepts Learned
- Routers in Make
- Filters (branch conditions)
- Manual data mapping
- Logging form data into Google Sheets
- Using Google Forms as the source for routing decisions

## Challenges
- Understanding nested field mapping
- Mapping values from Google Forms correctly
- Learning how Router conditions evaluate data
- Debugging scenarios when branches did not execute
